# WhatsApp Bot AI - Demo

## Sobre o Projeto Original

O WhatsApp Bot AI é um sistema completo de chatbot para WhatsApp desenvolvido em React com backend Node.js. O projeto original inclui:

- Interface de gerenciamento de conversas
- Sistema de autenticação
- Dashboard com estatísticas
- Configurações de chatbot
- Integração com APIs de IA
- Sistema de CRM integrado

## Adaptações para Demo

Esta versão demo foi criada para demonstrar a interface e funcionalidades principais sem a necessidade de backend ou autenticação:

### Arquivos Copiados e Adaptados:

1. **Interface Principal**: Baseada nos componentes React originais
   - `Conversations.jsx` → Interface de conversas
   - `ChatModal.jsx` → Modal de chat
   - `Sidebar.jsx` → Navegação lateral
   - `Navbar.jsx` → Barra de navegação

2. **Estilos**: Adaptados do Tailwind CSS original para CSS vanilla
   - Mantida a paleta de cores azul (#3b82f6)
   - Preservado o layout responsivo
   - Mantidos os efeitos de hover e transições

3. **Funcionalidades Simuladas**:
   - Chat interativo com respostas automáticas
   - Múltiplas conversas com dados mock
   - Indicadores de áudio e status online
   - Estatísticas de contatos únicos
   - Navegação entre seções

### Características Preservadas:

- ✅ Design moderno e profissional
- ✅ Interface responsiva
- ✅ Simulação de bot com respostas inteligentes
- ✅ Múltiplas conversas ativas
- ✅ Indicadores visuais (áudio, status, contadores)
- ✅ Navegação intuitiva
- ✅ Efeitos de digitação (typing indicator)

### Funcionalidades Demo:

- **Chat Interativo**: Digite mensagens e receba respostas automáticas do bot
- **Múltiplas Conversas**: Clique nas conversas da sidebar para alternar
- **Navegação**: Explore diferentes seções (Dashboard, Configurações)
- **Responsivo**: Funciona em desktop, tablet e mobile

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com flexbox e grid
- **JavaScript ES6+**: Interatividade e simulação
- **Google Fonts**: Tipografia Inter

## Como Usar

1. Abra o arquivo `index.html` em um navegador
2. Explore as diferentes conversas na sidebar
3. Digite mensagens no chat para interagir com o bot
4. Navegue pelas diferentes seções usando o menu lateral

## Fonte Original

Este demo foi adaptado do projeto WhatsApp Bot AI localizado em `Whatsapp bot. AI/frontend/`, preservando a essência da interface original enquanto remove dependências de backend e autenticação para criar uma experiência de demonstração fluida.
