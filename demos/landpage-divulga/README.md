# TTECH Landing Page Demo

## Overview

This is a static HTML adaptation of the TTECH landing page, originally built as a React application with TypeScript and Tailwind CSS. The demo preserves all original functionality and design while converting it to vanilla HTML, CSS, and JavaScript for portfolio demonstration purposes.

## Source Project Information

- **Original Project**: TTECH Launchpad (React + TypeScript + Tailwind CSS)
- **Source Location**: `landpage divulga/ttech-launchpad-main/`
- **Adaptation Date**: December 2024
- **Demo Purpose**: Portfolio demonstration of landing page development capabilities

## Files Copied and Adapted

### Core Files
- `index.html` - Main landing page (adapted from React components)
- `styles.css` - Complete CSS adaptation from Tailwind classes
- `script.js` - JavaScript functionality adapted from React hooks and components
- `hero-bg.jpg` - Hero section background image
- `public/favicon.ico` - Site favicon
- `public/robots.txt` - SEO robots file
- `public/placeholder.svg` - Placeholder image

## Development Notes

### Adaptation Process
1. **Component Analysis**: Analyzed React component structure
2. **Style Extraction**: Converted Tailwind classes to vanilla CSS
3. **Functionality Mapping**: Recreated React hooks in vanilla JavaScript
4. **Asset Integration**: Copied and optimized all media assets
5. **Demo Enhancement**: Added portfolio-specific navigation and features

### Key Challenges Solved
- Converting React component state to vanilla JavaScript
- Adapting Tailwind utility classes to custom CSS
- Maintaining responsive behavior without framework
- Preserving complex animations and interactions
- Ensuring cross-browser compatibility
